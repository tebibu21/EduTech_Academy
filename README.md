## ğŸ§­ Project Overview

**EduTech Academy** is designed to present an online learning platform offering modern, interactive courses.  
It includes:
- A visually appealing **Home** page.
- An informative **About** section.
- A **Courses** catalog grid with hover effects.
- A functional **Contact** page with a form and embedded map.

This project is ideal for learning **responsive web design**, **Bootstrap components**, and **semantic HTML**.

## ğŸ¨ Customization Tips
- To **change the theme color**, update the `.btn-primary`, `.bg-dark`, and `.text-primary` colors in `style.css`.
- Replace image URLs in HTML with your own image links.
- Add new courses by duplicating one of the `<div class="col-md-4">` blocks in `courses.html`.
- To modify the footer icons, update the Font Awesome icon classes (e.g., `fa-facebook`, `fa-instagram`).
